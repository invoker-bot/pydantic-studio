# Security Policy

## Supported Versions

pydantic-studio is currently in alpha. Security fixes target the latest
`main` branch and the next tagged release. Older untagged snapshots are not
supported.

## Reporting a Vulnerability

Do not report suspected vulnerabilities in public issues.

Use GitHub private vulnerability reporting for this repository when available.
If that is not available, open a minimal public issue asking for a private
coordination channel, but do not include exploit details, private data, or
reproduction steps in that public issue.

Please include the affected version or commit, a concise impact summary, and a
minimal reproduction once a private channel is established.
